UNIK(1) "Filters"

# NAME

unik - print duplicate lines, keeping original line order

# SYNOPSIS

*unik*

# DESCRIPTION

This is like “*sort* | *uniq*” without sorting the lines first.

# SEE ALSO

*sort*(1), *uniq*(1)
