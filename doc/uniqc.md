UNIQC(1) "Filters"

# NAME

uniqc - print unique lines, prefixing lines by the number of occurrences.

# SYNOPSIS

*uniqc*

# DESCRIPTION

This is like “*sort* | *uniq* _-c_” without sorting the lines first.

# SEE ALSO

*sort*(1), *uniq*(1)
