SORTC(1) "Filters"

# NAME

sortc - sort unique lines according to their number of occurrence

# SYNOPSIS

*sortc*

# DESCRIPTION

This is like “*sort* | *uniq* _-c_ | *sort* _-rn_” without the
initial sort.

# SEE ALSO

*sort*(1), *uniq*(1)
